# cal-app — ระบบคำนวณปรับชดเชยค่าตอบแทนพนักงานจ้าง อปท.

เว็บแอปคำนวณการปรับชดเชย / เลื่อนค่าตอบแทน และเงินตกเบิกของพนักงานจ้าง อปท.
(เชื่อมโยงเอกสาร 1–5 + สรุปรายบุคคล + แนวทางบันทึกบัญชี New e-LAAS)

เป็น **static site ล้วน** — ทำงานทั้งหมดในเบราว์เซอร์ฝั่งหน้าเว็บ
ไม่มี backend ไม่มีฐานข้อมูล ไม่เรียก API ภายนอกใด ๆ ข้อมูลที่กรอกจะอยู่ในหน่วยความจำของแท็บที่เปิดเท่านั้น
และผู้ใช้สามารถกด **บันทึกเป็นไฟล์ JSON** เพื่อเก็บไว้ และ **เปิดไฟล์** กลับมาใช้ภายหลังได้

> ลิงก์ภายนอกเดียวที่ใช้คือ Google Fonts (ฟอนต์ Sarabun / Bai Jamjuree) ซึ่งโหลดผ่าน CDN
> ถ้าเปิดแบบออฟไลน์ ระบบยังทำงานได้ครบ เพียงแต่จะใช้ฟอนต์สำรองของระบบแทน

## โครงสร้างไฟล์

```
cal-app/
├── index.html      # ตัวแอปทั้งหมด (HTML + CSS + JS ในไฟล์เดียว)
├── vercel.json     # ตั้งค่า deploy เป็น static site บน Vercel
├── .gitignore
└── README.md
```

## รันบนเครื่อง (Local)

เนื่องจากเป็น static HTML จะเปิดไฟล์ `index.html` ตรง ๆ ในเบราว์เซอร์ก็ได้
แต่แนะนำให้เสิร์ฟผ่าน local server เพื่อให้เหมือนตอน deploy จริง:

```bash
# ทางเลือกที่ 1 — Node (ไม่ต้องติดตั้งอะไร มากับ npx)
npx --yes serve -l 3000

# ทางเลือกที่ 2 — Python (ติดมากับเครื่องอยู่แล้ว)
python -m http.server 3000
```

แล้วเปิดเบราว์เซอร์ที่ **http://localhost:3000**

## Deploy บน Vercel

แอปนี้เป็น static site จึง deploy ได้ทันที ไม่ต้อง build:

1. **ผ่านเว็บ Vercel** — เชื่อมต่อ GitHub repo นี้ → Import → Framework Preset เลือก **Other** →
   เว้น Build Command และ Output Directory ว่างไว้ → Deploy
2. **ผ่าน CLI**
   ```bash
   npm i -g vercel
   vercel        # preview
   vercel --prod # production
   ```

ไฟล์ `vercel.json` กำหนดให้ Vercel เสิร์ฟไฟล์ตรง ๆ โดยไม่ต้อง build อยู่แล้ว
